## Java中的运算符

1. 赋值运算符 =

2. 扩展运算符

   > +=
   >
   > /=
   >
   > %=
   >
   > |=
   >
   > .....

3. 算数运算符

   > System.out.println(3.0/0);//Infinity
   >
   > System.out.println(10/0.0); //Infinity
   >
   > System.out.println(-10/0.0);//-Infinity
   >
   > System.out.println(-0/0.0);//NaN
   >
   > System.out.println(--0/0.0);//NaN
