## HashMap集合

键是哈希表结构，可以保证键的唯一性。

往map里放数据,遇到相同Key后面的会把前面的覆盖.


- Map集合遍历
  1. 根据键找值。获取所有键的集合，遍历键的集合，获取到每一个键，根据键找值。
  
  2. 根据键值对对象找键和值。
     - 获取所有键值对对象的集合
     - 遍历键值对对象的集合，获取到每一个键值对对象
     - 根据键值对对象找键和值
  
  3. 直接输出值
  
     ```
     Collection<E> values = map.values();遍历values
     ```
